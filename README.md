# 7-Zip Icon Pack
An extended set of icons for file formats that [7-Zip](https://www.7-zip.org/) can handle (in the style of different operating systems), as well as icons for the program itself (also in SVG format). All PSD files included.

## Unified Icon (for all Arcives)
Sometimes I notice that some people use the 7-Zip icon on their computers for all archives (the standard icon assignment by the system), so I decided to make one unified icon in the style of the program icon for all types of archives, as an option.
